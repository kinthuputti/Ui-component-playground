# Ui-component-playground

A collection of clean, modern, and reusable UI components featuring a sophisticated **glassmorphism** aesthetic. Designed for developers who want to add a premium, "frosted" look to their web projects without the overhead of heavy frameworks.

## 🎨 Aesthetic
* **Frosted Glass:** Utilizes `backdrop-filter: blur(8px)` for depth and elegance.
* **Minimalist:** Soft, subtle borders and transparent surfaces.
* **Responsive:** Built with a flexible CSS Grid system that adapts to any screen size.

## 🚀 Quick Start
Simply include `style.css` in your project and use the following markup structure:

```html
<div class="component-box">
  <h4>Component Name</h4>
  <div class="preview-content">
    </div>
  <button class="copy-btn">Copy Code</button>
</div>

🛠 Features
Zero-Dependency: Pure CSS—no JS frameworks required.

Easy Customization: Modify the background opacity or blur intensity in the .component-box class.

Responsive Grid: Automatically adjusts via grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)).
